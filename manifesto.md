# Manifesto: The Surrender of Sergeant Mac

## 🕊️ Artistic Statement
"Knockin' on Heaven's Door: The Surrender of Sergeant Mac" is a digital exploration of the boundary between combat and catharsis. By placing the user in the role of a silent observer or a compassionate superior, the project challenges players to use language as a de-escalation tool rather than a weapon. The transition from the physical weight of the M16 to the symbolic weight of the sheriff's badge represents the slow peeling away of the "soldier" persona to reveal the human underneath.

## 🔍 AI Transparency & Ethics
This project leverages generative AI not as a shortcut, but as a dynamic medium for real-time storytelling. Every interaction is unique, and the visual transformations (inpainting) serve as a metaphor for the permanence of Mac's emotional decisions.

### AI Model Registry
| Tool/API | Usage in Project |
| :--- | :--- |
| **Groq / Llama 3.3** | Powers Mac's emotional intelligence, dialogue, and state transitions. |
| **YOLO-World** | Provides the "eyes" of the system to identify targets for removal. |
| **OpenAI DALL-E 2** | Performs the real-time "erasure" of objects through generative inpainting. |
| **OpenAI DALL-E 3** | Generated the foundational visual character and environment assets. |
| **Suno AI** | Composed the ambient acoustic background loop to maintain tension. |
| **runwayml (SD)** | Serves as the high-fidelity local fallback for inpainting. |
